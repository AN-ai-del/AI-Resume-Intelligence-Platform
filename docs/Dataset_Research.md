# 📊 Dataset Research

# CareerLens AI

Version 1.0

---

# 🎯 Objective

The objective of this document is to evaluate potential datasets for CareerLens AI and select the most appropriate one for Version 1.

The chosen dataset should support:

- Resume parsing
- Skill extraction
- Job role prediction
- ATS analysis
- Future expansion into LLMs and AI Agents

---

# 📌 Dataset Requirements

The dataset should satisfy the following criteria:

- Resume text available
- Job role labels
- Multiple job categories
- Balanced classes
- Clean formatting
- Publicly available
- Suitable for Machine Learning
- Allows future improvements

---

# 📋 Evaluation Criteria

| Criteria | Importance |
