# 📦 CareerLens AI - Version 1 Scope

**Version:** 1.0

**Status:** Planning

---

# 🎯 Objective

Build the first working version of CareerLens AI capable of analyzing resumes and providing meaningful career insights using Machine Learning and Natural Language Processing.

The primary goal of Version 1 is to create a complete end-to-end AI application that demonstrates software engineering best practices and serves as the foundation for future versions.

---

# ✅ Must-Have Features

These features are required before Version 1 can be considered complete.

## Resume Processing

- Upload resume (PDF)
- Extract text from PDF
- Clean and preprocess text

---

## Resume Analysis

- Detect technical skills
- Calculate ATS compatibility score
- Identify missing skills
- Predict suitable job role
- Display confidence score

---

## Recommendations

- Resume strengths
- Resume weaknesses
- Improvement suggestions
- Recommended projects
- Recommended certifications

---

## User Interface

- Upload resume
- Interactive dashboard
- Charts and visualizations
- Clear recommendations
- Responsive layout

---

## Engineering

- Modular source code
- Logging
- Error handling
- Documentation
- Deployment

---

# ⭐ Nice-to-Have Features

If time permits, Version 1 may also include:

- Resume score breakdown
- Skill category visualization
- Downloadable report
- Resume statistics
- Resume word cloud

These features are optional and should not delay the release.

---

# ❌ Out of Scope

The following features will NOT be developed during Version 1.

## AI

- LLM integration
- AI chatbot
- RAG
- AI Agents

---

## Backend

- Authentication
- User accounts
- Database
- Cloud storage

---

## External Services

- LinkedIn integration
- Job scraping
- Email notifications
- Resume rewriting

---

## Enterprise Features

- Recruiter dashboard
- Multi-user support
- Admin panel

---

# 📈 Success Criteria

Version 1 will be successful if:

- Resume upload works reliably.
- Resume text extraction succeeds.
- Skills are detected correctly.
- Job role prediction works.
- ATS score is generated.
- Recommendations are meaningful.
- Dashboard is responsive.
- Application is deployed.
- Documentation is complete.
